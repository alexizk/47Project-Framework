# Glossary

- **Identity**: profile picture, wallpaper, lock screen
- **Tri-state checkbox**: checkbox with 3 states (No change / Off / On)
- **No change**: dash state; tool does not modify that setting
- **Snapshot**: JSON saved before apply; used to restore or undo
- **Portable mode**: store data next to the script in `IdentityKitData`
- **Enforced**: uses policy/system methods to increase reliability (may require admin + reboot)
