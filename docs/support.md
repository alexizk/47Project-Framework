# Support / Bug Reports

If you open an issue or ask for help, include the following. It saves a lot of back-and-forth.

---

## Must include

1. **What you tried**
   - which page
   - which buttons (Apply vs Quick Apply)
   - which options you changed

2. **Badges**
   - `Admin: Yes/No`
   - `Mode: Portable/Standard`

3. **Windows info**
   - Windows 10/11
   - version/build number (Settings → System → About)

4. **Log**
   - click **Copy log** and paste it
   - or attach the latest log file from the Logs folder

---

## Helpful extras

- If lock screen: mention whether you rebooted
- If a toggle: mention if it was No change / Off / On

---

## What not to include
- passwords, license keys
- private company domains
- personal documents
